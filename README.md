# A Counselling Website usign HTML, CSS & Javascript

This project is a fully responsive website for A Somebody Counselling, built as a portfolio piece to demonstrate HTML, CSS, and JavaScript skills. It includes multiple pages, interactive tools, and a clean, accessible design suitable for a professional counselling service. It was designed to add to a beginner technology and engineering portfolio, using a real world scenario to develop practical front-end development skills and gain experience building a structured and styled website from code for the first time. 

## Project Purpose

This project demonstrates:
- Front-end web development skills (HTML, CSS, JavaScript)
- Responsive design and layout techniques
- Integration of interactive JavaScript features
- Attention to user experience and accessibility
- Troubleshooting

## Pages Overview

### Homepage
A welcoming landing page featuring an introduction, professional qualifications, and a personal message. Includes a simple contact form and banner image, designed for readability and calm, professional aesthetics.

### About Page
Provides a detailed overview of A Somebody’s counselling approach, philosophy, and experience. Clear headings and responsive layout help clients understand the therapist’s style and specialisms.

### Services Page
Highlights counselling services with a grid layout, transparent fees, and practical details. Includes a two-column “What Therapy is Like” section with an image and explanatory text to clarify the therapy process.

### My Counselling Room Page
Showcases the therapy space with descriptive text and a photo gallery. Demonstrates the calm, private, and welcoming environment clients can expect.

### Resources Page
Offers mindfulness and wellbeing tools, including a breathing exercise and interactive candle-lighting game. Features a curated list of trusted mental health organisations and helplines, designed for easy access and reflection.

### Contact Page
A responsive contact page with a two-column layout: an image and a submission form. Includes essential contact information and ensures accessibility and usability.

## Features

- Fully responsive design for desktop, tablet, and mobile devices
- Grid and flexbox layouts for content and image alignment
- Interactive JavaScript tools: breathing exercise and candle-lighting game
- Consistent navigation and footer across all pages
- Accessible and readable typography with clear hierarchy

## Technologies Used

- HTML5
- CSS
- JavaScript (mindfulness games)
